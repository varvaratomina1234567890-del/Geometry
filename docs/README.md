# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

## Общее описание решения

Данный проект представляет собой простую библиотеку на Python для вычисления основных геометрических параметров круга и квадрата. Проект состоит из двух модулей, каждый из которых содержит функции для работы с определенной геометрической фигурой.

**Основной функционал:**
- Модуль `circle.py`: вычисление площади круга и длины окружности
- Модуль `square.py`: вычисление площади и периметра квадрата

Проект демонстрирует принципы модульности, документирования кода и может быть использован как учебный пример или как основа для более сложных математических библиотек.

## Описание функций

### Модуль `circle.py`

#### `area(r)`

Вычисляет площадь круга по заданному радиусу.

**Параметры:**
- `r` (int | float): Радиус круга. Должен быть неотрицательным.

**Возвращаемое значение:**
- `float`: Площадь круга.

**Исключения:**
- `ValueError`: Вызывается, если передан отрицательный радиус.

**Пример вызова:**
```python
import circle

circle_area = circle.area(5)
print(circle_area)```  # Выведет: 78.5
```

#### `perimetr(r)`

Вычисляет периметр круга по заданному радиусу.

**Параметры:**
- `r` (int | float): Радиус круга. Должен быть неотрицательным.

**Возвращаемое значение:**
- `float`: периметр круга.

**Исключения:**
- `ValueError`: Вызывается, если передан отрицательный радиус.

**Пример вызова:**
```python
import circle

circle_perimetr = circle.perimetr(5)
print(circle_perimetr)  # Выведет: 31,4
```
